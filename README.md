# 📰 Dragon News

A modern news web application built with **Next.js**, featuring category-based news, dynamic routing, and a fully responsive UI.

---

## 🚀 Features

- 📂 Category-based news filtering
- 📰 Dynamic news details page
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast performance with Next.js server components
- 🎨 Clean and modern UI with Tailwind CSS
- 🔗 Social links and sidebar widgets
- 🖼️ Optimized images using Next.js Image component

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- JavaScript
- React Icons
